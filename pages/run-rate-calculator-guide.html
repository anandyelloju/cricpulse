Cricket run rate calculator guide

Run rate shows how quickly a team is scoring. It is calculated as total runs divided by overs faced.

Current run rate

If a team scores 72 runs in 12 overs, the current run rate is 6.00. CricPulse updates this automatically during live cricket scoring.

Required run rate

In a chase, required run rate uses runs still needed divided by overs remaining. It helps batters understand how quickly they must score to reach the target.

Balls and partial overs

In cricket notation, 10.3 overs means 10 overs and 3 balls, not 10.3 decimal overs. A run rate calculator should convert balls correctly before dividing.
